# 🏥 MediVision Hospital Analytics Dashboard

## 📌 Project Overview

This project is an interactive Hospital Analytics Dashboard developed using Microsoft Power BI.

The dashboard helps analyze hospital performance through interactive KPIs, charts, and slicers, enabling quick decision-making.

---

## 📊 Dashboard Features

- Total Patients KPI
- Treatment Cost Analysis
- Doctor Performance
- Department Analysis
- Disease Analysis
- Insurance Analysis
- Gender Distribution
- Patient Status
- City-wise Analysis
- Interactive Filters (Slicers)

---

## 🛠 Tools Used

- Microsoft Power BI
- Power Query
- DAX
- Data Cleaning
- Data Visualization
